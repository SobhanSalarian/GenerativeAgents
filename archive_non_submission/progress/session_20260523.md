## 2. Composite Believability Formula — Weight Logic

Formula: `B = 0.30·C + 0.25·M + 0.25·P + 0.20·N`

| Weight | Dimension | Rationale |
|---|---|---|
| 0.30 | Behavioural consistency | Foundation — inconsistency undermines all other dimensions |
| 0.25 | Memory coherence | Primary ablation mechanism; no theoretical priority over planning |
| 0.25 | Planning plausibility | Primary ablation mechanism; equal to memory by design |
| 0.20 | Response naturalness | Weakest discriminator — fluent LLMs produce natural text regardless of architecture; ceiling-effect risk |

## 3. Sub-Dimension Internal Weights

These are not in the chapter (v3.2) and are not derived from literature — they are implementation judgment calls.

| Sub-dimension | Formula |
|---|---|
| Behavioural consistency | `0.45 × request_consistency + 0.35 × profile_alignment + 0.20 × cooperation_rate` |
| Memory coherence | `0.4 × mention_flag + 0.6 × embedding_relevance` |
| Planning plausibility | `0.4 × explicit_reference + 0.6 × embedding_alignment`; if LLM available: `0.5 × LLM + 0.5 × embedding` |
| Response naturalness | `0.6 × LLM_score + 0.4 × heuristic` (pure heuristic if LLM unavailable) |

**Pattern:** surface citation is consistently weighted less (0.4) than semantic alignment (0.6). The assumption: producing a reference token is easy; semantic relevance between the cited content and the actual decision requires genuine use of the cognitive module.

---

## 4. Academic Acceptability of the Weighting Approach

**Composite-level weights (0.30/0.25/0.25/0.20): acceptable.**
Judgment-based weights are standard in composite index construction provided they are stated, argued for, and sensitivity-tested. The chapter does all three. An examiner can disagree with the weights but cannot reject the approach on methodological grounds.

**Sub-dimension internal weights: currently not acceptable — fixable.**
The problem is not that they are judgment calls. The problem is they are *undisclosed* judgment calls. They affect results but the chapter says nothing about them. The standard for MRes requires disclosure, informal justification, and acknowledgement as a limitation — not empirical derivation.

## 5. Disclosure Paragraph Drafted for §X.4.5

To be inserted after the sensitivity-checks sentence in §X.4.5:

> The sub-dimension scores are themselves weighted composites. Behavioural consistency combines request consistency (0.45), persona-profile alignment (0.35), and cooperation rate (0.20), with request consistency receiving the highest weight as the most direct behavioural signal. Memory coherence and planning plausibility each apply a 0.4 / 0.6 split between an explicit-citation indicator and an embedding-based semantic relevance score; the higher weight on relevance reflects the judgement that citing a memory or plan token is a weaker signal than semantic alignment between the cited content and the decision context. Response naturalness applies a 0.6 / 0.4 split between an LLM-assisted naturalness judge and a heuristic proxy when the LLM judge is available, reverting to the heuristic alone otherwise. These sub-dimension weights are implementation judgements rather than literature-derived values. The composite-level sensitivity checks defined above partially bound this dependence, but the sub-weights themselves are not varied independently.
