Submission layout for the WISE 2026 paper

- Main manuscript source: ../Final_draft.tex
- Bibliography: ../bib/references.bib
- Figures: ../figures/
- Compiled PDF: ./paper.pdf

Suggested workflow:
1. Keep the main LaTeX source in the repository root for now.
2. Put your bibliography in bib/references.bib.
3. Store all figures in figures/.
4. Generate the PDF into submission/paper.pdf.
